ADVERSARIAL MACHINE LEARNING

Adversarial machine learning refers to the study of the vulnerabilities and defenses against adversarial attacks on machine learning models. Adversarial attacks involve intentionally manipulating or perturbing input data in order to deceive or mislead a machine learning model, leading to incorrect or unexpected outputs

Classification Factors

Robustness
Robustness in machine learning refers to the ability of a model to maintain its performance and generalization capabilities in the presence of various perturbations, such as noise in the input data, adversarial attacks, or distribution shifts.
Explainability
Explainability in machine learning refers to the ability to understand and interpret the decisions or predictions made by a machine learning model in a human-understandable manner.
Fairness
Fairness in machine learning refers to the concept of ensuring that machine learning models make unbiased and equitable decisions, treating individuals or groups fairly and without discrimination.
Uncertainity
Uncertainty in machine learning refers to the lack of complete confidence or knowledge about the predictions or decisions made by a machine learning model.
